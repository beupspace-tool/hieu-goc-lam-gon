Ở Level này, bạn sẽ hiểu cách phần mềm được tạo ra, sự khác biệt giữa các sản phẩm Claude, và thực hiện cuộc hội thoại đầu tiên với Claude Code. Đây là nền tảng để mọi thứ sau này đều có nghĩa.

8 Bài học trong Level 1

Đọc tuần tự từ đầu đến cuối. Bài 1.3 cần bạn cài đặt trên máy thật.

1.1
Chu kỳ phát triển phần mềm

Mọi phần mềm đều trải qua cùng một chu kỳ: Ý tưởng → Thiết kế → Phát triển → Kiểm thử → Triển khai → Bảo trì. Hiểu Claude Code đang giúp bạn ở bước nào.

⏱ ~6 phút
→
1.2
Claude Code vs Claude Chat vs API

Claude có 3 cách sử dụng khác nhau, phục vụ 3 mục đích khác nhau. Chọn đúng công cụ để hiệu quả gấp đôi.

⏱ ~5 phút
→
1.3
Cài đặt & Xác thực

Hướng dẫn từng bước: cài Node.js, cài Claude Code, xác thực tài khoản. Làm đúng một lần, dùng mãi mãi.

⏱ ~6 phút
→
1.4
Khái niệm cốt lõi

4 khái niệm nền tảng: Context Window, Tokens, Prompts, và Tools. Hiểu 4 khái niệm này = hiểu 90% cách Claude vận hành.

⏱ ~7 phút
→
1.5
Cuộc hội thoại đầu tiên

3 bài tập thực hành đơn giản để làm quen với Claude Code. Mục tiêu: tạo confidence, không phải ấn tượng.

⏱ ~6 phút
→
1.6
Claude Code như trợ lý vận hành

Claude Code không thay thế bạn — nó khuếch đại bạn. 5 vai trò CC có thể đóng trong doanh nghiệp nhỏ của bạn.

⏱ ~5 phút
→
1.7
Sai lầm thường gặp

6 sai lầm phổ biến của người dùng mới. Hiểu trước = tránh 80% frustration khi bắt đầu.

⏱ ~6 phút
→
1.8
Bài tập thực hành

3 bài tập có hướng dẫn: tổ chức folder, viết email chào hàng, và phân tích tài liệu. Hoàn thành để sẵn sàng sang Level 2.
