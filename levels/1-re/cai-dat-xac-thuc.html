Bài 1.3: Cài đặt & Xác thực

⏱ ~6 phút đọc Level 1 🖥️ Cần thực hành trên máy
💡
Nguyên lý

Cài đặt phần mềm giống như lắp ráp dụng cụ trước khi bắt đầu công việc. Làm đúng một lần, dùng mãi. Chọn cách phù hợp với bạn bên dưới — Desktop App (3 bước) hoặc Terminal (4 bước).

Có hai cách cài đặt Claude Code. Chọn cách phù hợp với bạn:

Cách A: Desktop App — Khuyến nghị cho người mới

Không cần cài gì thêm, không cần biết terminal. Tải app, đăng nhập, dùng ngay.

1
Tải Claude Code Desktop

Tải từ trang chính thức: macOS | Windows. Cài đặt như phần mềm bình thường — kéo vào Applications (Mac) hoặc Next → Install (Windows).

2
Đăng nhập

Mở app, đăng nhập bằng tài khoản Anthropic (tài khoản bạn dùng cho claude.ai). Xong — bạn đã sẵn sàng.

3
Bắt đầu làm việc

Kéo thư mục dự án vào app (hoặc bấm Open folder). Gõ câu đầu tiên:

Xin chào! Hãy nói cho tôi biết bạn là ai.

Nếu Claude trả lời — cài đặt thành công. Bạn có thể dùng / (slash commands), xem diff trực quan, và accept/reject thay đổi bằng nút bấm.

Cách B: Qua Terminal — Cho ai đã quen dòng lệnh

Xem hướng dẫn cài qua Terminal (4 bước)

Bước 1: Kiểm tra Node.js

Claude Code CLI chạy trên nền tảng Node.js — hãy nghĩ Node.js như động cơ xe: CC là chiếc xe, Node.js là động cơ bên trong. Mở Terminal và gõ:

node --version

Nếu thấy v20.11.0 hoặc tương tự — OK. Nếu command not found, tải từ nodejs.org (phiên bản LTS), cài như phần mềm bình thường.

Bước 2: Cài Claude Code

npm install -g @anthropic-ai/claude-code

npm là công cụ quản lý phần mềm đi kèm Node.js (như App Store cho phần mềm dòng lệnh). -g nghĩa là cài toàn hệ thống. Chờ 1-2 phút cho đến khi terminal hiện lại dấu nhắc $.

Bước 3: Xác thực

claude

Lần đầu tiên, Claude Code sẽ hiển thị một URL — copy, dán vào trình duyệt, đăng nhập bằng tài khoản Anthropic. Hoặc nhập API Key (dạng sk-ant-...) nếu có.

Bước 4: Kiểm tra

claude --version

Thấy số phiên bản là OK. Gõ claude, rồi thử: "Xin chào! Hãy nói cho tôi biết bạn là ai." Nếu Claude trả lời — thành công.

Xử lý lỗi thường gặp

Desktop App: Không đăng nhập được
Kiểm tra kết nối mạng. Đảm bảo bạn dùng đúng tài khoản Anthropic (tài khoản claude.ai). Nếu vẫn lỗi, thử đóng app hoàn toàn rồi mở lại.
Terminal: npm — command not found
Node.js chưa cài. Tải từ nodejs.org, cài xong mở Terminal mới (đóng cửa sổ cũ, mở lại).
Terminal: EACCES — permission denied
Thêm sudo trước lệnh: sudo npm install -g @anthropic-ai/claude-code. Nhập mật khẩu máy (không hiển thị khi gõ — bình thường).
Terminal: claude — command not found
Đóng Terminal và mở lại. Nếu vẫn lỗi, thử dùng Desktop App — đơn giản hơn và không cần cấu hình PATH.
⚠️
Lưu ý về chi phí

Claude Code sử dụng API tokens và tính phí theo lượng sử dụng. Kiểm tra pricing tại anthropic.com/pricing. Với plan Pro ($20/tháng), bạn được credit đủ cho hầu hết công việc thông thường. Nếu bạn chỉ thử nghiệm học tập, chi phí sẽ rất thấp — thường dưới $5/tháng ở giai đoạn đầu.

🔧
Thực hành

Hoàn thành cài đặt bằng Cách A hoặc Cách B. Khi Claude chào bạn và trả lời câu hỏi đầu tiên — chụp ảnh màn hình lại. Đó là khoảnh khắc bắt đầu hành trình mới.
