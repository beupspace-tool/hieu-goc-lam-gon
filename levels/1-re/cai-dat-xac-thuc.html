Bài 1.3: Cài đặt & Xác thực

⏱ ~6 phút đọc Level 1 🖥️ Cần thực hành trên máy
💡
Nguyên lý

Cài đặt phần mềm giống như lắp ráp dụng cụ trước khi bắt đầu công việc. Làm đúng một lần, dùng mãi. Bài này có 4 bước — hãy làm theo thứ tự, đừng bỏ qua bước nào.

Trước khi bắt đầu, hãy mở Terminal trên máy bạn. Nếu bạn chưa quen Terminal, hãy quay lại Bài 0.4 để ôn lại. Đây là bước duy nhất trong Level 1 mà bạn phải làm trên máy thật.

1
Kiểm tra Node.js

Claude Code chạy trên nền tảng Node.js — một "môi trường chạy" cho phép các chương trình JavaScript hoạt động ngoài trình duyệt. Hãy nghĩ Node.js như động cơ xe: CC là chiếc xe, Node.js là động cơ bên trong.

Mở Terminal và gõ lệnh sau, rồi nhấn Enter:

node --version

Nếu bạn thấy kết quả như v20.11.0 hoặc số phiên bản tương tự, Node.js đã được cài. Bước 1 hoàn thành.

Nếu terminal báo command not found, bạn cần cài Node.js:

  1. Vào nodejs.org bằng trình duyệt
  2. Tải về phiên bản LTS (Long-Term Support) — ổn định nhất
  3. Cài đặt như phần mềm bình thường (Next → Next → Install)
  4. Mở Terminal mới, kiểm tra lại bằng node --version
2
Cài Claude Code

Khi đã có Node.js, cài Claude Code chỉ cần một lệnh. Gõ vào Terminal:

npm install -g @anthropic-ai/claude-code

Giải thích từng phần:

  • npm — Node Package Manager, công cụ quản lý phần mềm đi kèm Node.js (như App Store cho phần mềm dòng lệnh)
  • install — lệnh cài đặt
  • -g — "global", cài đặt toàn hệ thống (dùng được từ bất kỳ thư mục nào)
  • @anthropic-ai/claude-code — tên chính xác của gói phần mềm Claude Code

Quá trình cài mất khoảng 1-2 phút. Terminal sẽ hiển thị các dòng log — đó là bình thường. Chờ cho đến khi terminal trở lại hiện dấu nhắc $.

3
Xác thực tài khoản

Sau khi cài xong, khởi động Claude Code bằng lệnh:

claude

Lần đầu tiên chạy, Claude Code sẽ yêu cầu xác thực. Có hai cách:

Cách 1 (Đơn giản hơn): Đăng nhập qua trình duyệt

Terminal sẽ hiển thị một URL. Copy URL đó, dán vào trình duyệt, đăng nhập bằng tài khoản Anthropic của bạn (tài khoản bạn dùng cho claude.ai). Sau khi đăng nhập thành công trên trình duyệt, quay lại Terminal.

Cách 2: Dùng API Key

Nếu bạn có API Key từ console.anthropic.com, bạn có thể nhập trực tiếp khi được hỏi. API Key có dạng sk-ant-....

Sau khi xác thực thành công, bạn sẽ thấy giao diện chào mừng của Claude Code trong Terminal.

4
Kiểm tra hoạt động

Kiểm tra phiên bản Claude Code đã cài:

claude --version

Bạn sẽ thấy số phiên bản như 1.x.x. Sau đó, khởi động một phiên làm việc:

claude

Claude sẽ chào bạn và sẵn sàng nhận lệnh. Thử gõ một câu đơn giản:

Xin chào! Hãy nói cho tôi biết bạn là ai.

Nếu Claude trả lời, cài đặt thành công hoàn toàn. Nhấn Ctrl+C (hoặc gõ /exit) để thoát khi xong.

Xử lý lỗi thường gặp

npm: command not found
Node.js chưa cài hoặc chưa được nhận dạng. Giải pháp: Cài Node.js từ nodejs.org, sau đó mở Terminal mới (đóng cửa sổ cũ, mở lại). Terminal cũ không nhận cập nhật PATH.
EACCES: permission denied
Không đủ quyền cài ở cấp global. Giải pháp trên Mac/Linux: thêm sudo trước lệnh: sudo npm install -g @anthropic-ai/claude-code. Nhập mật khẩu máy tính của bạn khi được hỏi (mật khẩu không hiển thị khi gõ — bình thường).
Authentication failed / Unauthorized
Tài khoản chưa được xác thực đúng. Giải pháp: Chạy claude lại, chọn phương thức đăng nhập qua trình duyệt (Browser login), đảm bảo bạn đăng nhập đúng tài khoản Anthropic. Nếu dùng API Key, kiểm tra key có bắt đầu bằng sk-ant- không.
claude: command not found (sau khi cài xong)
PATH chưa cập nhật. Giải pháp: Đóng Terminal và mở lại. Trên một số máy Mac, bạn cần thêm export PATH="$HOME/.npm-global/bin:$PATH" vào file ~/.zshrc hoặc ~/.bash_profile.

🔧
Thực hành

Hoàn thành cả 4 bước trên. Khi bạn thấy Claude chào bạn trong Terminal và trả lời câu hỏi đầu tiên — chụp ảnh màn hình lại. Đó là khoảnh khắc bắt đầu hành trình mới. Không phải mỗi ngày bạn có một "lần đầu tiên" như thế này.
