0.1
Phần mềm là gì?

Hiểu phần mềm là tập hợp chỉ dẫn cho máy tính — từ app Zalo đến hệ thống ATM. Nền tảng để hiểu mọi thứ còn lại.

Internet & Web

Internet là mạng lưới, web là một dịch vụ chạy trên đó. Hiểu server, browser, và cách một trang web thực sự hoạt động.

AI là gì? LLM là gì?

AI không phải robot biết suy nghĩ. Tìm hiểu cách LLM hoạt động, Claude là ai, và tại sao AI hữu ích nhưng không hoàn hảo.

Terminal — Công cụ giao tiếp bằng chữ

Cửa sổ đen với con trỏ nhấp nháy. Đừng sợ — terminal chỉ là cách gõ lệnh trực tiếp cho máy tính, mạnh mẽ hơn nút bấm.

"Code" là gì?

Code là ngôn ngữ viết chỉ dẫn cho máy tính. Bạn không cần viết code — Claude Code làm thay. Nhưng hiểu code giúp bạn ra lệnh tốt hơn.

Claude Code không phải chatbot. Đây là trợ lý có thể đọc file, viết file, và thao tác trên máy tính của bạn — như nhân viên IT siêu giỏi.
