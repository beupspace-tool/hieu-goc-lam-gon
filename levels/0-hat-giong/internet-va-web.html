Bài 0.2: Internet & Web

⏱ ~4 phút đọc Level 0
💡
Nguyên lý

Kiến trúc web ảnh hưởng trực tiếp đến chi phí vận hành và quyết định kỹ thuật của bạn. Khi website bạn chậm, nguyên nhân có thể là hosting kém, thiếu CDN, code không tối ưu, hoặc database quá tải. Biết phân biệt giúp bạn hỏi đúng câu hỏi — và không bị agency "thổi giá" khi không cần thiết.

Client-server — mọi thứ bạn dùng đều hoạt động theo mô hình này

Mỗi khi bạn mở website, app, hay dịch vụ cloud, đều có một cuộc trao đổi xảy ra: client (thiết bị của bạn) gửi yêu cầu, server (máy chủ của nhà cung cấp) xử lý và trả về kết quả. Đây là nền tảng của toàn bộ web hiện đại.

Tại sao điều này quan trọng với doanh nghiệp? Vì nó quyết định dữ liệu của bạn nằm ở đâu. Khi bạn dùng Google Sheets — dữ liệu nằm trên server của Google ở Mỹ. Khi bạn dùng MISA cloud — dữ liệu nằm trên server của MISA tại Việt Nam. Khi bạn dùng phần mềm on-premise — dữ liệu nằm trong văn phòng bạn.

Domain, hosting, CDN — ba thứ bạn phải hiểu khi có website

Domain (tên miền)

Địa chỉ website của bạn — ví dụ congtyabc.vn. Bạn thuê quyền sử dụng tên miền từ nhà đăng ký (Godaddy, Namecheap, hay các nhà cung cấp Việt như VNPT, Matbao). Chi phí: ~200.000–500.000đ/năm cho tên miền .vn. Tên miền và hosting là hai thứ riêng biệt — bạn có thể mua domain ở một nơi và hosting ở nơi khác.

Hosting (nơi đặt server)

Máy chủ chạy website của bạn 24/7. Ba lựa chọn chính:

  • Shared hosting (~50.000–200.000đ/tháng): nhiều website chung một server. Rẻ nhưng chậm khi hàng xóm traffic cao.
  • VPS (~200.000–1.000.000đ/tháng): server riêng ảo hoá. Phù hợp cho SME có traffic ổn định.
  • Cloud hosting (AWS, Google Cloud, Azure): trả theo dùng, tự động scale. Chi phí biến động theo traffic — phù hợp khi có spike đột biến như sale 11/11.

CDN (Content Delivery Network)

Mạng lưới server phân tán toàn cầu. Khi khách hàng ở Hà Nội truy cập website, họ lấy file từ server gần nhất thay vì từ server gốc ở TP.HCM. Kết quả: tải nhanh hơn 30–60%. Cloudflare có gói miễn phí hoạt động tốt cho hầu hết SME Việt Nam.

🔧
Chẩn đoán website chậm

Nếu website của bạn chậm, dùng pagespeed.web.dev (miễn phí của Google) để kiểm tra. Công cụ này chỉ ra đúng nguyên nhân: hình ảnh chưa nén, thiếu CDN, code CSS/JS quá nặng, hay server response time kém. Báo cáo này cũng là tài liệu để bạn làm việc với agency — thay vì nghe "cần nâng cấp toàn bộ", bạn biết chính xác cần fix gì.

HTTP vs HTTPS — và tại sao HTTPS là bắt buộc

HTTP là giao thức truyền dữ liệu giữa browser và server. HTTPS là phiên bản mã hoá — dữ liệu được mã hoá trên đường truyền, không ai "nghe lén" được. Google phạt SEO website HTTP, Chrome hiển thị cảnh báo "Not secure", và khách hàng mất tin tưởng. Chứng chỉ SSL (HTTPS) hiện nay miễn phí qua Let's Encrypt — không có lý do gì để website doanh nghiệp vẫn chạy HTTP.

⚠️
Rủi ro dữ liệu với SaaS web

Mọi dịch vụ SaaS bạn dùng — Google Workspace, Notion, Hubspot — đều lưu dữ liệu trên server của họ theo mô hình client-server. Điều này có nghĩa: khi tài khoản bị khoá, bị hack, hoặc nhà cung cấp ngừng hoạt động, bạn có thể mất quyền truy cập dữ liệu. Giải pháp thực tế: thiết lập backup tự động định kỳ ra file local hoặc cloud storage riêng của bạn (Google Drive, S3). Đây không phải phòng ngừa thừa — đây là quản lý rủi ro cơ bản.
