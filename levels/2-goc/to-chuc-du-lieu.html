Bài 2.7: Tổ chức dữ liệu

⏱ ~6 phút đọc Level 2
💡
Nguyên lý

Data có giá trị khi có cấu trúc. Claude Code có thể biến data lộn xộn thành thông tin có ý nghĩa — nhưng bạn cần cho CC biết bạn cần gì. Giống như kho hàng: CC là người kho giỏi nhất thế giới, nhưng bạn phải nói muốn kiểm đếm gì và theo tiêu chí nào.

CSV — Bạn thân của doanh nghiệp nhỏ

CSV (Comma-Separated Values) là định dạng dữ liệu đơn giản nhất và được CC xử lý tốt nhất. Nếu bạn có dữ liệu trong Excel, hãy lưu lại thành .csv trước khi yêu cầu CC xử lý.

Ví dụ một file CSV đơn giản:

email,ho_ten,so_dien_thoai,tinh_thanh,tong_mua_hang
nguyen.an@gmail.com,Nguyễn An,0901234567,Hà Nội,5200000
tran.bich@yahoo.com,Trần Bích,0912345678,TP.HCM,3800000
le.cuong@gmail.com,Lê Cường,0923456789,Đà Nẵng,7100000

Format này đơn giản, dễ đọc, và CC có thể xử lý file CSV lên đến hàng nghìn dòng mà không gặp vấn đề gì.

Dọn dẹp dữ liệu

Đây là việc tốn nhiều thời gian nhất khi quản lý data. CC có thể làm thay bạn:

Đọc file contacts.csv. Thực hiện các bước sau:
1. Loại bỏ tất cả dòng có email trùng lặp
   (giữ lại dòng có ngày gần nhất nếu có cột ngày)
2. Chuẩn hóa cột "ho_ten" về dạng Title Case
   (Ví dụ: "NGUYEN VAN AN" → "Nguyễn Văn An")
3. Loại bỏ khoảng trắng thừa ở đầu/cuối mỗi ô
4. Lưu file đã dọn vào contacts-clean.csv
Báo cáo: đã xóa bao nhiêu dòng trùng, sửa bao nhiêu tên.
Đọc file products.csv.
Tìm và liệt kê:
- Sản phẩm có cột "giá" bị thiếu hoặc để 0
- Sản phẩm có tên bị trùng lặp
- Dòng nào có dữ liệu bất thường (giá âm, tồn kho âm)
Không sửa file, chỉ báo cáo vấn đề tìm được.

Phân tích đơn giản

CC có thể tính toán và phân tích dữ liệu mà không cần bạn biết Excel hay công thức phức tạp:

Đọc file sales-2026.csv (cột: ngày, sản_phẩm, số_lượng, đơn_giá).
Tính:
1. Tổng doanh thu theo từng tháng
2. Top 5 sản phẩm bán chạy nhất (theo số lượng)
3. Ngày bán hàng tốt nhất trong năm
Trả kết quả dưới dạng bảng Markdown.
Phân tích file khach-hang.csv:
- Phân nhóm khách hàng theo tỉnh thành
- Tính tổng và trung bình mua hàng mỗi nhóm
- Xác định nhóm khách hàng có giá trị cao nhất

Chuyển đổi format

Data từ nhiều nguồn khác nhau thường có format khác nhau. CC giúp bạn chuyển đổi nhanh:

Chuyển đổi file products.csv thành bảng Markdown
để paste vào báo cáo. Chỉ lấy 3 cột:
tên sản phẩm, giá bán, tồn kho.
Đọc file config.json và chuyển thành file
CSV để dễ chỉnh sửa bằng Excel.
Mỗi key-value pair thành một dòng.

Kết hợp nhiều file

Khi data nằm ở nhiều file, CC có thể merge lại theo logic bạn chỉ định:

Kết hợp 2 file:
- customers.csv (cột: email, ho_ten, tinh_thanh)
- orders.csv (cột: email_khach, ma_don, gia_tri, ngay)

Tạo file customers-with-orders.csv mới gồm:
email, ho_ten, tinh_thanh, so_don_hang, tong_gia_tri
(một dòng cho mỗi khách hàng, tổng hợp tất cả đơn).
Đọc tất cả file CSV trong folder /reports/monthly/
và tạo file annual-summary.csv tổng hợp
tất cả doanh thu theo từng tháng trong năm 2025.
⚠️
Cảnh báo

Trước khi CC thực hiện bất kỳ thao tác nào làm thay đổi file dữ liệu quan trọng (xóa dòng, sửa giá trị, merge file), hãy tạo bản backup trước. Yêu cầu CC lưu kết quả ra file mới thay vì ghi đè file gốc. Ví dụ: contacts-clean.csv thay vì ghi đè contacts.csv.

🔧
Thực hành

Tạo file products.csv với 10 sản phẩm mẫu (columns: ten, gia, danh_muc, ton_kho). Thêm vài dòng dữ liệu lộn xộn: tên viết hoa không nhất quán, một vài giá bị thiếu, 2 sản phẩm trùng tên. Sau đó yêu cầu CC: (1) Sắp xếp theo giá từ thấp đến cao, (2) Tìm sản phẩm có tồn kho thấp hơn 10, (3) Tính giá trung bình theo từng danh mục.
