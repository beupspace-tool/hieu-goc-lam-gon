# Hiểu Gốc — Làm Gọn

> Claude Code cho người vận hành

Giáo trình thực hành Claude Code thiết kế theo nguyên lý first-principles. Từ người chưa biết gì về công nghệ đến thành thạo vận hành doanh nghiệp với AI.
